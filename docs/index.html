Sources

Surpass is a multi-platform graphical password manager with focus on simplicity. The source code is free (as in "liberty") and is designed to be concise and simple.


  • Securely store any secret e.g. passwords, pass phrases, pin codes, etc.
  • Small footprint, less than 40MB. Can fit on any memory device. Does not need installation or any third party library, completely self-contained.
  • System tray support.
  • Clipboard support. If a secret is copied to clipboard it's automatically cleared after 30 seconds.
